# Sistema de Farmacia

## Estructura del Proyecto
- `/backend` - API REST con Node.js y Express
- `/frontend` - Interfaz de usuario con React

## Tecnologías
- Backend: Node.js, Express, PostgreSQL
- Frontend: React, Vite, Material-UI

## Instalación

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
